# Math Trainer App   

**Math Trainer** — це веб-застосунок для тренування арифметичних навичок (додавання, віднімання, множення, ділення). Проєкт розроблено в рамках курсу "Компонентно-орієнтоване програмування".

##  Функціонал  
- **Ігрова механіка:** Генерація прикладів з урахуванням обраного рівня складності.  
- **Рівні складності:**  
  - **Easy:** Тільки додавання.  
  - **Medium:** Додавання, віднімання, табличка множення.  
  - **Hard:** Всі операції, включаючи ділення та великі числа.  
- **Глобальний стан:** Використання Zustand для збереження налаштувань та результатів.  
- **Персистенція:** Автоматичне збереження прогресу в `localStorage` браузера.  
- **Статистика:**  
  - Особистий профіль з рекордами.  
  - Глобальна таблиця результатів з фільтрацією та сортуванням.  
- **UI Компоненти:** Розроблені та документовані в Storybook.

##  Технологічний стек  
- **Core:** React, Vite  
- **State Management:** Zustand (з middleware persist)  
- **Routing:** React Router Dom v6  
- **Forms:** React Hook Form + Yup Validation  
- **Documentation:** JSDoc, Storybook  
- **Legal:** Cookie Consent (GDPR compliance)

##  Конфігурація та Встановлення  
Для запуску проєкту вам знадобиться встановлений **Node.js** (версія 16+).  

### Клонування репозиторію:  
```bash
git clone https://github.com/JaroslavAnd/KOP.git
cd KOP
npm install
```

###  Запуск проєкту (Dev mode)

```bash
npm run dev
```

Відкрийте у браузері:  
http://localhost:5173
