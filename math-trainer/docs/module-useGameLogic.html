Module: useGameLogic

Користувацький хук для управління основною логікою математичної гри. Відповідає за таймер, генерацію прикладів, перевірку відповідей та збереження результатів. * @returns {Object} Об'єкт зі станом гри та функціями управління.
Properties:
Name Type Description
timeLeft number Час, що залишився до кінця гри (в секундах).
score number Поточний рахунок гравця.
currentProblem Object Об'єкт поточного прикладу.
Properties
Name Type Description
text string Текст прикладу для відображення.
answer number Правильна математична відповідь.
userAnswer string Поточна відповідь, яку ввів користувач.
isGameOver boolean Прапорець, що вказує на завершення гри.
isGameActive boolean Прапорець, що вказує, чи гра зараз триває.
setUserAnswer function Функція для оновлення введеної відповіді користувача.
handleAnswerSubmit function Функція для перевірки відповіді та нарахування балів.
restartGame function Функція для перезапуску гри з поточними налаштуваннями.
